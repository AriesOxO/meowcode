   - 优先级：高

   - 解决方案：自动复制 example 文件

   - 影响：Agent 可能卡住
   - 优先级：中
   - 解决方案：添加格式验证和修复建议

## 📚 学习收获

### 技术层面
1. **Agent 设计原理**: 理解了 LLM → Parse → Execute → Feedback 循环
2. **抽象层设计**: 学会了用工厂模式实现多模型支持
3. **API 集成**: 掌握了 OpenAI SDK 的使用和自定义 base_url
4. **Windows 编码**: 解决了 Python 在 Windows 上的编码问题

### 工程层面
1. **模块化设计**: 职责清晰，易于维护和扩展
2. **测试驱动**: 从 API → 逻辑 → 流程 的分层测试
3. **文档优先**: 边做边写，避免事后补文档
4. **版本管理**: 规范的 Git 提交信息

### 认知层面
1. **不要过度设计**: Python vs Go 的选择，开发效率优先
2. **迭代式开发**: 从 MVP 开始，逐步完善
3. **成本意识**: DeepSeek 性价比分析
4. **用户视角**: 从 CLI 开始，而不是复杂的 GUI

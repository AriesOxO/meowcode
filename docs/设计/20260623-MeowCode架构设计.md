# MeowCode Agent 架构设计方案

## 一、项目定位

### 目标
构建一个轻量级、易扩展的 AI 编码助手，核心特性：
- 🎯 前期专注 DeepSeek，后期支持多模型
- 💰 价格友好（DeepSeek 成本优势）
- 🚀 跨平台（Windows/macOS/Linux）
- 📦 易部署（单文件打包）

### 与 Claude Code 的差异化
- ✅ 更轻量（核心功能优先）
- ✅ 多模型支持（不绑定单一提供商）
- ✅ 开源透明（可自定义扩展）
- ✅ 国内友好（DeepSeek 无需代理）

---

## 二、编程语言选择

### 候选语言对比

| 语言 | 优点 | 缺点 | 打包方案 | 推荐度 |
|------|------|------|---------|--------|
| **Python** | 生态成熟、开发快、AI 库丰富 | 打包大、启动慢 | PyInstaller, Nuitka | ⭐⭐⭐⭐⭐ |
| **Go** | 打包小、启动快、跨平台强 | AI 生态弱、开发慢 | 原生编译 | ⭐⭐⭐⭐ |
| **Rust** | 性能最强、内存安全 | 学习曲线陡、开发慢 | 原生编译 | ⭐⭐⭐ |
| **TypeScript** | 前端生态好、Electron 集成 | 需 Node 运行时 | Electron, pkg | ⭐⭐⭐ |

### 推荐：Python

**理由**：
1. ✅ **最快 MVP**：1-2 周即可完成核心功能
2. ✅ **AI 生态最强**：OpenAI/Anthropic SDK 都是 Python 优先
3. ✅ **代码简洁**：正则、文件操作、subprocess 都是标准库
4. ✅ **打包可接受**：PyInstaller 单文件 20-40MB
5. ✅ **学习资源多**：遇到问题容易找解决方案

**打包方案**：
- **阶段 1（开发期）**：直接 `python agent.py`
- **阶段 2（测试期）**：`PyInstaller --onefile` 打包
- **阶段 3（优化期）**：`Nuitka` 编译（体积更小、速度更快）

## 三、多模型架构设计

### 3.1 抽象层设计

```python
# models/base.py
from abc import ABC, abstractmethod
from typing import List, Dict

class BaseLLMProvider(ABC):
    """LLM 提供商基类"""
    
    @abstractmethod
    def query(self, messages: List[Dict]) -> str:
        """查询模型"""
        pass
    
    @abstractmethod
    def get_model_info(self) -> Dict:
        """获取模型信息（名称、价格、上下文长度）"""
        pass
    
    @abstractmethod
    def validate_api_key(self) -> bool:
        """验证 API Key 是否有效"""
        pass
```

### 3.2 具体实现

```python
# models/deepseek.py
from openai import OpenAI
from .base import BaseLLMProvider

class DeepSeekProvider(BaseLLMProvider):
    def __init__(self, api_key: str):
        self.client = OpenAI(
            api_key=api_key,
            base_url="https://api.deepseek.com"
        )
        self.model = "deepseek-v4-flash"
    
    def query(self, messages: List[Dict]) -> str:
        response = self.client.chat.completions.create(
            model=self.model,
            messages=messages
        )
        return response.choices[0].message.content
    
    def get_model_info(self) -> Dict:
        return {
            "name": "DeepSeek V4 Flash",
            "price": {"input": 0.1, "output": 0.2, "unit": "¥/M tokens"},
            "context": 64000
        }
```

```python
# models/zhipu.py
from zhipuai import ZhipuAI

```

```python
# models/anthropic.py
from anthropic import Anthropic
from .base import BaseLLMProvider

class AnthropicProvider(BaseLLMProvider):
    def __init__(self, api_key: str):
        self.client = Anthropic(api_key=api_key)
        self.model = "claude-sonnet-4.5"
    
    def query(self, messages: List[Dict]) -> str:
        # 注意：Anthropic 不支持 system 在 messages 里
        system_msg = None
        filtered_msgs = []
        for msg in messages:
            if msg["role"] == "system":
                system_msg = msg["content"]
            else:
                filtered_msgs.append(msg)
        
        response = self.client.messages.create(
            model=self.model,
            max_tokens=4096,
            system=system_msg,
            messages=filtered_msgs
        )
        return response.content[0].text
```

### 3.3 工厂模式

```python
# models/factory.py
from typing import Dict
from .base import BaseLLMProvider
from .deepseek import DeepSeekProvider
from .zhipu import ZhipuProvider
from .anthropic import AnthropicProvider

PROVIDERS: Dict[str, type] = {
    "deepseek": DeepSeekProvider,
    "zhipu": ZhipuProvider,
    "anthropic": AnthropicProvider,
}

def create_provider(provider_name: str, api_key: str) -> BaseLLMProvider:
    """创建 LLM 提供商实例"""
    if provider_name not in PROVIDERS:
        raise ValueError(f"Unknown provider: {provider_name}")
    
    return PROVIDERS[provider_name](api_key)
```

### 3.4 使用示例

```python
# agent.py
from models.factory import create_provider

# 配置文件或环境变量读取
provider = create_provider("deepseek", api_key="sk-xxx")

# 统一接口调用
response = provider.query(messages)
```

## 四、价格对比分析

### 4.1 主流模型价格（2026-06）

| 提供商 | 模型 | 输入价格 | 输出价格 | 上下文 | 推荐场景 |
|--------|------|---------|---------|--------|---------|
| **DeepSeek** | v4-flash | ¥0.1/M | ¥0.2/M | 64K | Agent 首选 ⭐⭐⭐⭐⭐ |
| DeepSeek | v4-pro | ¥1.0/M | ¥2.0/M | 64K | 复杂推理 |
| 智谱 GLM | glm-4-flash | ¥0.1/M | ¥0.1/M | 128K | 性价比高 ⭐⭐⭐⭐ |
| 智谱 GLM | glm-4-plus | ¥50/M | ¥50/M | 128K | 高质量输出 |
| Anthropic | Claude Sonnet 4.5 | $3/M | $15/M | 200K | 最强能力 ⭐⭐⭐ |
| OpenAI | GPT-4o | $2.5/M | $10/M | 128K | 平衡 |

### 4.2 实际使用成本估算

**场景：修复一个 Bug**

典型流程：
1. 理解需求（1次调用，1K tokens）
2. 搜索相关代码（2次调用，各2K tokens）
3. 分析问题（1次调用，3K tokens）
4. 生成修复代码（1次调用，输出1K tokens）
5. 验证和调整（2次调用，各2K tokens）

总计：输入约 12K tokens，输出约 3K tokens

| 模型 | 单次成本 | 100次成本 |
|------|---------|----------|
| DeepSeek v4-flash | ¥0.0018 | ¥0.18 |
| GLM-4-flash | ¥0.0015 | ¥0.15 |
| Claude Sonnet 4.5 | $0.075 (¥0.54) | $7.5 (¥54) |

**结论**：
- ✅ **DeepSeek 性价比最高**：国内 Agent 场景
- ✅ **GLM-4-flash**：备选方案，上下文更长
- ⚠️ **Anthropic/OpenAI**：高端场景，成本 30-50 倍

### 4.3 定价策略建议

**如果做商业化**：

| 用户等级 | 模型选择 | 定价策略 |
|---------|---------|---------|
| 免费版 | DeepSeek v4-flash | 100 次/月免费 |
| 个人版 | DeepSeek + GLM 可选 | ¥9.9/月（1000次） |
| 专业版 | 全模型可选 | ¥49/月（不限次数） |
| 企业版 | 私有部署 | 面议 |

## 五、跨平台打包方案

### 5.1 阶段规划

#### 阶段 1：开发期（Week 1-4）
**目标**：核心功能实现

```bash
# 直接运行
python agent.py

# 依赖管理
pip install -r requirements.txt
```

**发布形式**：
- GitHub Release 源码 + README
- 用户自行安装 Python 运行

#### 阶段 2：Beta 测试（Week 5-8）
**目标**：单文件打包

```bash
# PyInstaller 打包
pyinstaller --onefile --name meowcode agent.py

# 输出
dist/
  ├── meowcode.exe  (Windows, ~30MB)
  ├── meowcode      (Linux, ~25MB)
  └── meowcode      (macOS, ~25MB)
```

**优点**：
- ✅ 用户无需安装 Python
- ✅ 双击即运行
- ✅ 体积可接受（20-40MB）

**缺点**：
- ⚠️ 启动稍慢（1-2秒）
- ⚠️ 杀毒软件可能误报

#### 阶段 3：正式发布（Week 9+）
**目标**：优化体积和速度

**方案 A：Nuitka（推荐）**
```bash
nuitka --standalone --onefile agent.py

# 优势
- 体积减小 30-50%
- 启动速度提升 2-3 倍
- 更像原生应用
```

**方案 B：分发方式**
- Windows: `.exe` + Installer (NSIS/Inno Setup)
- macOS: `.app` + DMG
- Linux: AppImage / Snap / Flatpak

### 5.2 技术选型对比

| 方案 | 体积 | 启动速度 | 兼容性 | 开发成本 | 推荐度 |
|------|------|---------|--------|---------|--------|
| **PyInstaller** | 30MB | 慢(1-2s) | 高 | 低 | ⭐⭐⭐⭐⭐ (初期)|
| **Nuitka** | 15MB | 快(0.3s) | 高 | 中 | ⭐⭐⭐⭐⭐ (优化) |
| **py2exe** | 25MB | 慢 | 仅Windows | 低 | ⭐⭐ |
| **cx_Freeze** | 35MB | 慢 | 中 | 中 | ⭐⭐⭐ |

### 5.3 完整打包脚本

```python
# build.py
import PyInstaller.__main__
import platform
import shutil

def build():
    system = platform.system()
    
    args = [
        'agent.py',
        '--onefile',
        '--name', f'meowcode-{system.lower()}',
        '--icon', 'assets/icon.ico' if system == 'Windows' else 'assets/icon.icns',
        '--add-data', 'config:config',  # 打包配置文件
        '--hidden-import', 'openai',
        '--hidden-import', 'anthropic',
        '--console',  # 或 --windowed（GUI版本）
    ]
    
    PyInstaller.__main__.run(args)
    
    # 清理临时文件
    shutil.rmtree('build', ignore_errors=True)
    print(f"✅ Build completed: dist/meowcode-{system.lower()}")

if __name__ == '__main__':
    build()
```

### 5.4 CI/CD 自动构建

```yaml
# .github/workflows/build.yml
name: Build Release

on:
  push:
    tags:
      - 'v*'

jobs:
  build:
    strategy:
      matrix:
        os: [ubuntu-latest, windows-latest, macos-latest]
    
    runs-on: ${{ matrix.os }}
    
    steps:
      - uses: actions/checkout@v3
      
      - name: Set up Python
        uses: actions/setup-python@v4
        with:
          python-version: '3.11'
      
      - name: Install dependencies
        run: |
          pip install -r requirements.txt
          pip install pyinstaller
      
      - name: Build
        run: python build.py
      
      - name: Upload artifact
        uses: actions/upload-artifact@v3
        with:
          name: meowcode-${{ matrix.os }}
          path: dist/*
```
